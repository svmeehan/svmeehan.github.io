About Me

Seamus Meehan

Currently: Studying a HDip in Computing at Dublin Institute of Technology.

This site was developed for the Web and User Interface Design Module. It uses Bootsrap for responsivness

I am a highly analytical individual with experience doing scientific grade data analysis. My main work to date has been the development of an automated pipeline for analysis of astronomical images and scripts for fitting a model to the results. This work was done using python along with the modules NumPy, SciPy and MatPlotLib. I have excellent verbal and written communication skills having given talks at international conferences along with teaching students from a non science background in an astronomy elective. I have a keen interest in machine learningtechniques and have read a number of books on the subject.
